# Python File Management System

A command-line file manager built using Python that allows users to create, read, update, and delete files while exploring the folder structure.

## Features
- Create files
- Read files
- Update files
- Delete files
- Explore folder structure

## Technologies Used
- Python
- pathlib module

## How to Run

1. Clone the repository

git clone https://github.com/navneetsharma212006/Python-file-management-system.git

2. Go inside the project

cd Python-file-management-system

3. Run the program

python src/FileHandling.py
